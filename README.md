# Big Data Analytics in Pharmacovigilance

This repository contains academic materials and research summaries focused on the application of **Big Data Analytics, Artificial Intelligence (AI), and Machine Learning (ML)** in **Pharmacovigilance (PV)**.  
It was created as part of a **seminar / academic assignment** in the domain of **Big Data Analytics**.

---

## 📌 Project Overview

Pharmacovigilance plays a critical role in ensuring drug safety by detecting, assessing, and preventing adverse drug reactions (ADRs).  
With the explosion of healthcare data from sources such as **Electronic Health Records (EHRs), spontaneous reporting systems, social media, and real-world evidence**, big data analytics has become a transformative tool in modern PV systems.

This repository consolidates:
- Research papers
- Structured summaries
- Seminar reports
- Comparative analysis of recent literature
